# data-traffic
AppSheetで作ったデータ通信量の管理アプリです。
随時、データ通信の残量を登録することで残量の推移をチャートで確認できます。

# 日毎の集計
- 登録日時を日付に整形
- 日毎の最小値（残りデータ量）、最大値（使用データ量）をバーチャル項目に追加
- 最小値または最大値以外の行を非表示にする
